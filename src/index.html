¿Qué es el desarrollo fullstack?

La profesión de desarrollo web es una de las más demandadas por la industría. Tener conocimientos de Frontend está genial, pero conocer todo el stack de tu proyecto, hace tener una visión más completa y mejores oportunidades laborales. Hay múltiples lenguajes y frameworks y dependiendo del tipo de empresa se utilizan unos u otros, pero un stack que domina gran parte del mercado es el conocido como MERN, por sus siglas de MongoDB, Express, React y Node.js

  • React.js: Framework de JavaScript, creado y utilizado por Facebook y muchas otras empresas: (Airbnb, Netflix, Instagram, Uber, Eventbrite,...) con el que podemos crear interfaces de usuario reactivas de una manera sencilla y escalable basada en componentes.
  • Node.js: Entorno de ejecución que nos permite programar en JavaScript para el lado del servidor. Muy común en aplicaciones real-time y para el desarrollo de APIs REST.
  • Express: Framework muy ligero de Node.js que nos ayuda a crear servidores web y servicios con Node.js ahorrando líneas de código y que podemos configurar mediante middlewares.
  • MongoDB: Base de datos NoSQL para almacenar toda la información que genere nuestra aplicación. Su integración con Node.js es excelente y su no necesidad de esquemas nos permite desarrollar rápidamente.

Con estos conocimientos, tendrás la oportunidad de acceder a un gran número de puestos de trabajo como Frontend Engineer, o Fullstack Engineer

Qué aprenderás en la Fullstack JS Academy

El programa tiene una duración de 12 semanas, donde con un grupo reducido de personas veremos lo siguiente:

  • HTML/CSS: Aprender maquetación y un buen uso de estilos CSS es crucial para iniciarse en el desarrollo web.
  • JavaScript: Veremos el lenguaje de programación por excelencia de la web, desde cero hasta las últimas actualizaciones de ECMAScript.
  • React: Una vez conozcamos JavaScript nos pondremos manos a la obra con el framework más demandado y utilizado para crear SPAs.
  • Server Side Render: Lleva tu SPA más allá con renderizado desde el servidor
  • API REST con Node.js: Crea el backend de tus aplicaciones web con JavaScript. Securizada con Json Web Tokens.
  • Uso de bases de Datos, con MongoDB
  • Testing con Jest y Cypress
  • Despliegue en Cloud

Todo esto acompañado de manera online, con un mentor que te guiará, te corregirá y aclarará tus dudas día a día, con reuniones grupales e individuales.
