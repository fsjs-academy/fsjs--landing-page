¿Qué vas a aprender?

En esta academia se aprende haciendo.

Durante 8 semanas, semanalmente tendrás al menos una clase teórico/práctica sobre la que trabajarás, ya sea solucionando retos, ejercicios o el desarrollo de un proyecto.

El objetivo de este programa formativo es que aprendas las skills necesarias para construir aplicaciones web, tanto desde la capa Frontend, como el Backend, basado en APIs y bases de datos.

Al terminar y graduarte de este programa:

  • Conocerás JavaScript, el lenguaje de programación dominante en la web.
  • Implementarás sitios webs con HTML y CSS
  • Convertirás en dinámica tu página web utilizando React y Redux.
  • Programarás el backend de tu aplicación utilizando Node.js y Express.
  • Desarrollarás un API REST siguiendo buenas prácticas e implementando autenticación con JWT.
  • Implementarás tests para asegurar el funcionamiento de tus aplicaciones.
  • Aplicarás técnicas de Server-Side Rendering para mejorar el rendimiento y el SEO de tus desarrollos.
  • Podrás aplicar a ofertas de trabajo para perfiles más completos y de mayor nivel, cuyos salarios (en España) empiezan en 30.000€ anuales.

Módulos de la formación

  • HTML/CSS
    Aprender maquetación y un buen uso de estilos CSS es crucial para iniciarse en el desarrollo web.
  • JavaScript
    Veremos el lenguaje de programación por excelencia de la web, desde cero hasta las últimas actualizaciones de ECMAScript.
  • React
    Una vez conozcamos JavaScript nos pondremos manos a la obra con el framework más demandado y utilizado para crear SPAs.
  • Server Side Render
    Lleva tu SPA más allá con renderizado desde el servidor.
  • API REST con Node.js
    Crea el backend de tus aplicaciones web con JavaScript. Securizada con JSON WebTokens.
  • Bases de Datos
    Almacena de manera persistente los datos generados por tu aplicación en una base de datos. Usaremos MongoDB y PostgreSQL.
  • Testing con Jest y Cypress
    Asegúrate de que tu aplicación funciona como esperas, desarrollando test unitarios y de integración.
  • Despliegue en Cloud
    Aprende a desplegar a producción tu aplicación para ser consumida por todo el mundo.
  • Buenas prácticas
    De nada sirve saber todo esto si no aplicamos buenas prácticas y tenemos un código organizado y limpio.

    Te guiaremos profesionalmente, con revisiones de tu portfolio, y perfiles de GitHub y en Linkedin.
