¿Qué vas a aprender?

En esta academia se aprende haciendo.

Durante 8 semanas, semanalmente tendrás al menos una clase teórico/práctica sobre la que trabajarás, ya sea solucionando retos, ejercicios o el desarrollo de un proyecto.

El objetivo de este programa formativo es que aprendas las skills necesarias para construir aplicaciones web, tanto desde la capa Frontend, como el Backend, basado en APIs y bases de datos.

Al terminar y graduarte de este programa:

  • Conocerás JavaScript, el lenguaje de programación dominante en la web.
  • Implementarás sitios webs con HTML y CSS
  • Convertirás en dinámica tu página web utilizando React y Redux.
  • Programarás el backend de tu aplicación utilizando Node.js y Express.
  • Desarrollarás un API REST siguiendo buenas prácticas e implementando autenticación con JWT.
  • Implementarás tests para asegurar el funcionamiento de tus aplicaciones.
  • Aplicarás técnicas de Server-Side Rendering para mejorar el rendimiento y el SEO de tus desarrollos.
  • Podrás aplicar a ofertas de trabajo para perfiles más completos y de mayor nivel, cuyos salarios (en España) empiezan en 30.000€ anuales.

Módulos de la formación

  • HTML/CSS
    Aprender maquetación y un buen uso de estilos CSS es crucial para iniciarse en el desarrollo web.
  • JavaScript
    Veremos el lenguaje de programación por excelencia de la web, desde cero hasta las últimas actualizaciones de ECMAScript.
  • React
    Una vez conozcamos JavaScript nos pondremos manos a la obra con el framework más demandado y utilizado para crear SPAs.
  • Server Side Render
    Lleva tu SPA más allá con renderizado desde el servidor.
  • API REST con Node.js
    Crea el backend de tus aplicaciones web con JavaScript. Securizada con JSON WebTokens.
  • Bases de Datos
    Almacena de manera persistente los datos generados por tu aplicación en una base de datos. Usaremos MongoDB y PostgreSQL.
  • Testing con Jest y Cypress
    Asegúrate de que tu aplicación funciona como esperas, desarrollando test unitarios y de integración.
  • Despliegue en Cloud
    Aprende a desplegar a producción tu aplicación para ser consumida por todo el mundo.
  • Buenas prácticas
    De nada sirve saber todo esto si no aplicamos buenas prácticas y tenemos un código organizado y limpio.

Preguntas Frecuentes

  • ¿Qué conocimientos se requieren para acceder a ésta formación?
    Esta formación está pensada para personas que tienen conocimientos de programación (variables, bucles, condicionales, lógica, etc...) pero que tienen poca o ninguna experiencia en el desarollo web. Si tienes conocimientos muy avanzados de Frontend y/o de Backend, esta formación puede resultarte poco útil.
  • ¿Se necesita un computador/laptop muy avanzado
    No es necesario un equipo de última generación. Con que puedas tener un editor de código y un navegador es suficiente. Mi equipo personal tiene 7 años y funciona bien para estos temas. Lo que si necesitarás es conexión a internet, ya que la formación es completamente en remoto.
  • ¿Es un bootcamp Fulltime? ¿Cuánto tiempo he de dedicarle?
    El contenido para consumir será aproximadamente de 2h a la semana. Despues, el desarrollo de tu proyecto final, dependerá de ti y lo que puedas dedicar. Estimamos que con unas 5h-10h semanales (menos de 2h al día) podrás seguirlo perfectamente.
  • ¿El coste del curso es en Euros o Dólares?
    El precio son 1995 euros, que es el equivalente a $2170 USD. Puedes pagar en tu moneda local si te es más sencillo, envía un correo a team@fullstackjs.academy con tu caso y lo solucionamos.
  • ¿Que significa "más impuestos, si aplica"?
    Los impuestos dependen del pais de residencia del comprador. Si resides en Europa, se añadirá el % de IVA (Impuesto sobre el Valor Añadido) en el caso de España es un 21%. Si eres autonomo o empresa y estás registrado como operador intracomunitario, el IVA no se aplicará. De igual manera, si resides fuera de Europa, no se aplica ningún impuesto.
  • ¿Puedo solicitar factura?
    Claro, en el momento de la compra, te llegará un correo con el recibo, facturado desde Teachable, la plataforma que utilizamos para el contenido del curso y pasarela de pago.
  • ¿Puedo pagar de otra forma que no sea tarjeta de débito/crédito?
    Si, puedes pagar con PayPal y si lo deseas, también con Bitcoin. Envíanos un mensaje a team@fullstackjs.academy con tu caso.
  • ¿Se puede pagar a plazos?
    Si, si prefieres pagar en 2 plazos de 1000€ (o el equivalente en tu moneda), puedes hacerlo. Envíanos un mensaje a team@fullstackjs.academy con tu caso.
  • ¿Cómo es el funcionamiento de la formación
    Durante el tiempo de la formación (8 semanas) cada semana se subirá contenido nuevo relacionado con uno de los módulos a la plataforma. El contenido serán videos y texto con teoría y casos prácticos. Despues, durante la semana trabajarás en tu proyecto final, aplicando cada semana lo aprendido en el módulo. Tendrás soporte por un grupo de Slack Privado y sesiones 1 a 1 con el tutor para solucionar dudas más concretas y mentoría.
  • ¿Cuándo empieza?
    Esta edición empezará el 11 de Mayo, auqnue en la plataforma ya tienes contenido "pre-work" para configurar tu equipo de desarrollo.
