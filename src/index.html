El perfil de Desarrollador Fulltack es el más demandado

El desarrollo Web Fullstack abarca desde el Frontend, donde se utiliza HTML/CSS y JavaScript, junto con algún framework, siendo React.js uno de los más utilizados y demandados por las empresas. El Backend puede tener múltiples sabores, en nuestro caso apostamos por JavaScript con Node.js para utilizar el mismo lenguaje a lo largo de todo el stack.

Este perfil es áltamente demandado, no porque seas experto/a tanto en Frontend como en Backend, si no porque así tienes una visión más amplia de todo el flujo de la aplicación, su arquitectura y la interconexión entre todas las capas.

Contenidos que verás en la Fullstack JS Academy

En esta academia se aprende haciendo. Tendremos un poco de contenido teórico pero lo aplicaremos en seguida en forma de código. Todo el contenido es impartido de manera online. No importa donde vivas, no tienes que desplazarte a ningún sitio. Sólo necesitas tu ordenador y conexión a internet.

El contenido será prácticamente personalizado, ya que será exclusivo para 5 personas.

Cada semana tendremos varias sesiones, tanto grupales como individuales, donde veremos todo el siguiente contenido:

  • HTML/CSS
    Aprender maquetación y un buen uso de estilos CSS es crucial para iniciarse en el desarrollo web.
  • JavaScript
    Veremos el lenguaje de programación por excelencia de la web, desde cero hasta las últimas actualizaciones de ECMAScript.
  • React
    Una vez conozcamos JavaScript nos pondremos manos a la obra con el framework más demandado y utilizado para crear SPAs.
  • Server Side Render
    Lleva tu SPA más allá con renderizado desde el servidor.
  • API REST con Node.js
    Crea el backend de tus aplicaciones web con JavaScript. Securizada con JSON WebTokens.
  • Bases de Datos
    Almacena de manera persistente los datos generados por tu aplicación en una base de datos. Usaremos MongoDB y PostgreSQL.
  • Testing con Jest y Cypress
    Asegúrate de que tu aplicación funciona como esperas, desarrollando test unitarios y de integración.
  • Despliegue en Cloud
    Aprende a desplegar a producción tu aplicación para ser consumida por todo el mundo.
  • Buenas prácticas
    De nada sirve saber todo esto si no aplicamos buenas prácticas y tenemos un código organizado y limpio.
