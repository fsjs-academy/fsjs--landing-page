¿En qué consiste la fullstack JS Academy?

El programa tiene una duración de 12 semanas, donde con un grupo reducido de personas veremos lo siguiente:

  • HTML/CSS
    Aprender maquetación y un buen uso de estilos CSS es crucial para iniciarse en el desarrollo web.
  • JavaScript
    Veremos el lenguaje de programación por excelencia de la web, desde cero hasta las últimas actualizaciones de ECMAScript.
  • React
    Una vez conozcamos JavaScript nos pondremos manos a la obra con el framework más demandado y utilizado para crear SPAs.
  • Server Side Render
    Lleva tu SPA más allá con renderizado desde el servidor.
  • API REST con Node.js
    Crea el backend de tus aplicaciones web con JavaScript. Securizada con Json Web Tokens.
  • Bases de Datos
    Almacena de manera persistente los datos generados por tu aplicación en una base de datos. Usaremos MongoDB y PostgreSQL.
  • Testing con Jest y Cypress
    Asegúrate de que tu aplicación funciona como esperas, desarrollando test unitarios y de integración.
  • Despliegue en Cloud
    Aprende a desplegar a producción tu aplicación para ser consumida por todo el mundo.
  • Buenas prácticas
    De nada sirve saber todo esto si no aplicamos buenas prácticas y tenemos un código organizado y limpio.

Todo esto acompañado de manera online, con un mentor que te guiará, te corregirá y aclarará tus dudas día a día, con reuniones grupales e individuales.
